(constant) config :Object

Objeto de configuração em tempo de execução da aplicação. Cada propriedade é resolvida em tempo de execução preferindo valores expostos no objeto global window.__ENV__ (se presente) e, caso contrário, usando as variáveis de build process.env. Qualquer propriedade pode ser undefined se não for fornecida pelo ambiente.
Type:
  • Object
Properties:
Name Type Description
apiUrl string | undefined URL base da API. Prefere window.__ENV__.API_URL ou window.__ENV__.REACT_APP_API_URL, caso contrário process.env.REACT_APP_API_URL.
loginUnico string | undefined Host/flag do LoginUnico. Obtido de window.__ENV__.REACT_APP_LOGIN_UNICO ou process.env.REACT_APP_LOGIN_UNICO.
clientId string | undefined Identificador do cliente OAuth/OpenID. Obtido de window.__ENV__.REACT_APP_CLIENT_ID ou process.env.REACT_APP_CLIENT_ID.
redirectUri string | undefined URI de redirecionamento OAuth do cliente. Obtido de window.__ENV__.REACT_APP_REDIRECT_URI ou process.env.REACT_APP_REDIRECT_URI.
logoutUri string | undefined Endpoint/URI de logout. Obtido de window.__ENV__.REACT_APP_LOGOUT_URI ou process.env.REACT_APP_LOGOUT_URI.
apiRedirectUri string | undefined URI de redirecionamento da API backend (se diferente). Obtido de window.__ENV__.REACT_APP_API_REDIRECT_URI ou process.env.REACT_APP_API_REDIRECT_URI.

Access/guestAuth(REACT_APP_API_URL, guestName)

Autentica usuário convidado na API Libraskê, realizando requisição post na API. Caso seja realizada com sucesso, as credenciais de usuário obtidas são armazenadas e o usuário é direcionado para o jogo. Caso a requisição falhe, é exibida uma mensagem de erro ao usuário. As variáveis de ambiente necessárias para a operação são lidas do arquivo de configuração src/config.js.
Parameters:
Name Type Description
REACT_APP_API_URL string Endpoit da API Libraskê (config.apiUrl).
guestName string Nome do usuário que deseja se autenticar como convidado.
